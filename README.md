# SilksongFarsightMod
This mod enables the display of the game completion percentage on the save-profile screen without requiring the Farsight item.

<img width="858" height="724" alt="Farsight" src="https://github.com/user-attachments/assets/3e229277-8119-41ba-8048-59b72cfd9372" />

**Installation process:**

1) Install BeepInEx 5 (follow the install instructions described on its page). You can get it from [Nexus](https://www.nexusmods.com/hollowknightsilksong/mods/26)ï»¿ or from [GitHub](https://github.com/BepInEx/BepInEx/releases).

2) After the BeepInEx 5 installation, make sure that the BepInEx.dll file exists in the game installation directory.
By default it will be under:
(YourDrive):\Program Files (x86)\Steam\steamapps\common\Hollow Knight Silksong\BepInEx\core

3) Download and extract SilksongFarsightMod.zip and place the SilksongFarsightMod.dll inside BepInEx/plugins directory. 
By default it will be under:
(YourDrive):\Program Files (x86)\Steam\steamapps\common\Hollow Knight Silksong\BepInEx\plugins
